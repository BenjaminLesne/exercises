EN:
Creation of a movie Dashboard:
- Header
- Logo + Link to list page
  
- Home page
- Image of a film you want in the background
- List of 4 films of the moment
  
- Listings
- Tabs with different categories (Action | Animation | Adventure | Comedy | Documentary)
- Listing of films related to the selected category
- Favourites
- Add/delete
- Show wishlist icon on movie list
- Prerequisites
- Creation of Base components (ui/ux) (BaseTabs, BaseCard...)

Api to use: https://developers.themoviedb.org/3/getting-started/introduction

Techno:

* Vue3, Pinia

Plus :
- Typescript
- Tailwind

For the design, you can take inspiration from Netflix or let your imagination run wild.

Once you've finished, create a public git repo and send me the link.
